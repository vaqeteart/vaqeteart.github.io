Orgmode编辑文档

厌倦了markdown的格式调整，
markdown降低了学习的成本，但是由于过于简单，对于结构稍微复杂一点的文档，还是有些力不从心。
orgmode和asciidoc是一个不错的选择，但是，目前由于计划在Emacs下工作，所以暂时还是学习orgmode来编写。
如果markdown和orgmode打架的话，它一定不愿意这篇文章的诞生。
开始还是用markdown记录的，markdown功不可没落，暂时还缺少不了它，所以用它来记录orgmode的安装和基本的配置过程。
之后的内容，或许将会用orgmode来写了。

orgmode安装与加载：

  • 下载：

    $wget -c http://orgmode.org/org-8.3.1.tar.gz

  • 配置插件路径

    $tar -xzvf org-8.3.1.tar.gz -C ~/.emacs.d/installed

  • 配置orgmode编译

    $cd ~/.emacs.d/installed/org-8.3.1
$vim local.mk
# Where local software is found
prefix = /usr/share
localprefix = ~/.emacs.d/plugins

# Where local lisp files go.
lispdir= $(localprefix)/emacs/site-lisp/org

# Where local data files go.
datadir = $(localprefix)/emacs/etc/org

# Where info files go.
infodir = $(prefix)/info

    这里，为了便于迁移到其他机器，相关信息用localprefix来存放。

  • 编译安装：

    $make install-lisp install-etc

    这里编译配置和安装参考：http://orgmode.org/worg/dev/org-build-system.html
    根据参考资料，因为没有texi2pdf和makeinfo可能会导致install-info安装失败，所以不用`make install

    如果想要按照的话，那么先apt-get install texinfo, 这样会在系统上安装 makeinfo和texi2pdf命令。

  • 加载插件
    .emacs文件需要间接或直接包含的内容：
    (add-to-list 'load-path "~/.emacs.d/plugins/emacs/site-lisp/org")
    (require 'org-install)

  • 切换orgmode
    输入：M-x org-mode
    或直接打开：*.org文件。

  • 查看org-mode版本号：
    切换到org-mode模式之后，输入：M-x org-version
